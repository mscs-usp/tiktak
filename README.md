para usar o TikTak em seu projeto Maven
```
    <repository>
	  <id>my.mvn.repo</id>
	  <url>https://raw.github.com/tiktak-project/tiktak/mvn-repo/</url>
	  <!-- use snapshot version -->
	  <snapshots>
	     <enabled>true</enabled>
	      <updatePolicy>always</updatePolicy>
	   </snapshots>
	</repository>
```
